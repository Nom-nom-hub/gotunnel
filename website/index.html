Get Started in Minutes

Install GoTunnel and start tunneling in under 5 minutes

# Windows (winget)
winget install GoTunnel.Gotunnel

# macOS (Homebrew)
brew install gotunnel/tap/gotunnel

# Manual installation
git clone https://github.com/ogrok/gotunnel.git
cd gotunnel
go build -o gotunnel-server ./cmd/server
go build -o gotunnel-client ./cmd/client
